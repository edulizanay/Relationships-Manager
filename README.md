# Relationships Manager

A full-stack application for managing personal and professional relationships.

## Structure

- `frontend/` - React TypeScript frontend
- `backend/` - Node.js Express backend

## Getting Started

### Frontend
```bash
cd frontend
npm install
npm start
```

### Backend
```bash
cd backend
npm install
npm run dev
```
